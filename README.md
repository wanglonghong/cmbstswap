# CombustSwap Factory

# Local Development

The following assumes the use of `node@>=10`.

## Install Dependencies

`yarn`

## Compile Contracts

`yarn compile`

## Run Tests

`yarn test`

## Deploy to ganache

`yarn develop`

`yarn migrate_dev`

## Deploy to testnet

`yarn migrate_test`

## Deploy to mainnet

`yarn migrate_bsc`

NOTE: You need to replance operator address, mnemonics, dev_address and fee_setter_address with your own ones when to deploy to blockchain.
